  CkiLePatron — Guide d'utilisation
═══════════════════════════════════════════════════


── INSTALLATION ────────────────────────────────────

  1. Téléchargez CkiLePatron.exe depuis GitHub.

  2. Placez-le dans le dossier de votre choix.

  Aucune installation requise — Python et toutes
  les dépendances sont inclus dans l'exe.


── LANCEMENT ───────────────────────────────────────

  Double-cliquez sur : CkiLePatron.exe

  L'application démarre et le navigateur s'ouvre
  automatiquement sur http://127.0.0.1:5000

  Astuce : clic droit sur CkiLePatron.exe
           > Envoyer vers > Bureau (créer un raccourci)


── EXTENSION NAVIGATEUR ────────────────────────────

  L'extension permet d'importer un patron depuis
  le site Klafoutis en un seul clic.

  Au premier lancement, un dossier "extension"
  apparaît automatiquement à côté de l'exe.

  Pour l'installer dans Chrome ou Edge :

  1. Ouvrez la page des extensions :
       Chrome : chrome://extensions
       Edge   : edge://extensions

  2. Activez le "Mode développeur"
     (interrupteur en haut à droite).

  3. Cliquez sur "Charger l'extension non empaquetée".

  4. Sélectionnez le dossier "extension"
     situé à côté de CkiLePatron.exe.

  L'icône CkiLePatron apparaît alors dans la barre
  du navigateur.


── VOS PATRONS ─────────────────────────────────────

  Vos patrons sont stockés dans le dossier :
     patrons\

  Ce dossier est créé automatiquement à côté
  de CkiLePatron.exe au premier lancement.

  Format du nom de dossier :
     NOM - description - MARQUE - CIBLE
     ex : Blouse Margot - Col lavallière - Orageuse - Femme
